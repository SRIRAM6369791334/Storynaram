# Character Type Enum

## Purpose
Defines character role, archetype, and type enumerations.

## Character Roles (Narrative)
| Value | Description |
|-------|-------------|
| protagonist | Main protagonist |
| antagonist | Main antagonist |
| deuteragonist | Second major character |
| tritagonist | Third major character |
| love-interest | Romantic interest |
| mentor | Teacher or guide |
| confidant | Trusted friend/advisor |
| foil | Character foil |
| comic-relief | Comic relief character |
| narrator | Story narrator |

## Character Archetypes
| Value | Description |
|-------|-------------|
| hero | Classic hero |
| anti-hero | Morally ambiguous hero |
| villain | Classic villain |
| tragic-villain | Sympathetic villain |
| byronic-hero | Brooding, rebellious hero |
| everyman | Relatable ordinary person |
| sage | Wise mentor figure |
| trickster | Cunning, deceptive character |
| guardian | Protective figure |
| shadow | Dark reflection of the hero |
