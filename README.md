# Storynaram

Story Operating System — v2.1.0

A monorepo for the Storynaram platform.

## Prerequisites

- Node.js 22+
- pnpm 9+

## Getting Started

```bash
pnpm install
pnpm dev
```

## Project Structure

- apps/ — API, CLI, Worker
- packages/ — Core libraries and modules

## Scripts

- pnpm dev — Start API in watch mode
- pnpm build — Build all packages
- pnpm typecheck — TypeScript type checking
- pnpm lint — ESLint
- pnpm test — Run all tests
