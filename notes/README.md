# Notes Directory

## Purpose
The author's notebook. All unstructured and semi-structured notes, ideas, observations, and commentary related to the project.

## Responsibility
Provides a catch-all space for author thoughts â€” brainstorming notes, observations, reminders, ideas-in-progress, and any written material that doesn't belong in formal directories.

## Naming Convention
- **Files**: {note_id}.md or {note_id}.json
- **Structure**: Flat or topic-based subdirectories
- **IDs** follow config/id_rules.json

## Contents
- Brainstorming notes
- Idea fragments and concepts
- Writing observations
- Character development notes
- Plot idea notes
- World-building notes
- Magic system ideas
- Scene ideas and snippets
- Revision notes
- Author commentary
- To-do lists and reminders
- Meeting and discussion notes

## Future Expansion
- Note tagging and categorization
- Full-text note search
- Note-to-entity linking
- Note versioning
- Collaborative note editing
- Note export and sharing

## Relationships
- **All directories** notes can reference any domain
- **Research/** formal research outputs from notes
- **Reviews/** review feedback and notes
- **Prompts/** prompt testing notes
- **Memory/decisions** decision records from notes
- **Logs/** creation logs reference note activity
